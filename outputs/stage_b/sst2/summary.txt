INTERPRETATION NOTES
========================================================================
  * SST-2 is the negative-contrast dataset in this study. Fragmentation features are expected to be weaker predictors for sentiment classification than for math or QA tasks. Lower F1 on SST-2 is scientifically anticipated.
  * λ=0.50 is the safe default threshold for deployment.
  * λ=0.58 was tuned on OOF predictions (train set only, test set unseen).
  * For a deployment gate, prefer lower λ to maximise recall on FAILURE (lower FNR).
  * ROC-AUC and PR-AUC are threshold-independent — use them to compare models.
  * PR-AUC is especially informative when failure class is imbalanced.
  * F1_macro is the primary target metric; goal F1 > 0.75 stated for full system.
